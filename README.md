# 👾 Alien Invasion

Классическая аркадная игра, написанная на Python с использованием библиотеки Pygame. Управляй кораблём, уничтожай флот пришельцев и набирай очки. С каждым уровнем игра становится сложнее!


## 🚀 Особенности

- Полноценное игровое меню (кнопки «Начать», «Продолжить», «Выход»)
- Система паузы (клавиша Esc)
- Звуковые эффекты и фоновая музыка
- Прогрессия сложности: с каждым уровнем пришельцы движутся быстрее, а их ряды увеличиваются
- Подсчёт очков и рекорд (сохраняется до закрытия игры)
- Визуальное отображение оставшихся жизней (иконки кораблей)
- Возможность запуска как из исходного кода, так и в виде standalone `.exe` файла (Windows)

## 🛠 Технологии

- Python 3.14
- Pygame CE (Community Edition) — для графики, звука и игровой логики
- PyInstaller — для сборки исполняемого файла
